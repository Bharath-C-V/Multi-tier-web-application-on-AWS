# Multi-Tier Web Application on AWS

## Overview
This project demonstrates a multi-tier web application deployed on AWS using Docker, Kubernetes, Terraform, and GitHub Actions.

## Prerequisites
- AWS account
- AWS CLI installed and configured
- Terraform installed
- Docker installed
- Kubernetes CLI (kubectl) installed
